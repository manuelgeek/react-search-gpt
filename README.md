# React Serach GPT with Vite

This is a sample app that uses GPT API to display user search content. The content need to be formated to a redable UI for user

## Libraries and Stack

- https://github.com/remarkjs/react-markdown
- https://daisyui.com/
- Tailwind
- TypeScript
- Open API


## Set Up

run `yarn` 
then `yarn dev`
